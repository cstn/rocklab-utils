# Rocklab utils

> Utilities for Javascript front-end projects with React and Redux

## Requirements

- node version 14
- npm version 6
- React >= 16.8
- React redux

## Install

```shell script
npm install @rocklab/utils
```

## Development

### Setup

```shell script
git clone https://github.com/cstn/rocklab-utils.git
npm install
npm run lerna:bootstrap
```

### Commands

Install all dependencies

```shell script
npm install
npm run lerna:bootstrap
```

Check own code conventions

```shell script
npm run lint
```

Autoformat code

```shell script
npm run prettier
```
